## About the Project
This is my **personal portfolio website**, built to showcase my projects, skills, and experience as a computer science student.  
It highlights my journey in **web development**, featuring a clean UI and smooth navigation.

---

## Technologies Used
- **React.js (Vite)** — Frontend framework for fast development  
- **HTML5** — Structure of the web pages  
- **CSS3** — Styling and responsive design  
- **JavaScript (ES6+)** — Dynamic interactivity  
- **Surge** — Free static hosting for deployment

## ⚙️ Setup and Installation

To run this project locally:

```bash
# Clone the repository
git clone https://github.com/Zeeshan-Khan1/My-Portfolio.git

# Go into the folder
cd My-Portfolio

# Install dependencies
npm install

# Run the development server
npm run dev
